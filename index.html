Git es un software para la gestion a nivel local de las diferentes versiones de un proyecto.

Ventajas de usar git

  1. Control de versiones
  2. Integración con Visual
  3. Facilidad para crear ramas y mezclarlas
  4. Agiliza el flujo de trabajo

Principales comandos git

Comando Explicacion
git add Añade el cambio
git commit añadimos el cambio al stage
git push subimos los cambios a github
git checkout para crear una rama
git pull para actualizar esa rama a lo maximo, como por ejemplo el master
